Какая одежда нужна с собой?
Спортивная одежда, обувь. Для обеспечения личной безопасности рекомендуется взять с собой теплую одежду
В КАКОЙ СЕЗОН ПРОХОДЯТ КАТАНИЯ?
Январь - декабрь
КАК ОБЕСПЕЧЕНА БЕЗОПАСНОСТЬ ПРИ ПРОКАТЕ КВАДРОЦИКЛОВ?
Профессиональные инструкторы, современная импортная техника.
БУДЕТ ЛИ РЯДОМ ИНСТРУКТОР?
На протяжении всего маршрута катания, Вас будет сопровождать инструктор на отдельном квадроцикле. Он покажет лучшие места и приложит все усилия, чтобы поездка была насыщенной и запоминающейся
ЕСТЬ ЛИ ОГРАНИЧЕНИЯ ПО ВОЗРАСТУ?
Для управления квадроциклом допускаются лица достигшие 18 летнего возраста. Дети младше 16 лет катаются только вместе с взрослыми. Для катания с родителями допускаются дети от 4х лет
